## Conclusion

For the current benchmark cases, the extra pressure buffer is modest relative to 32 GiB RTX 5090 memory.
The largest tracked case is `perf_1gpu` and uses about `0.2456 GiB` for three pressure buffers, adding about `0.0819 GiB` over the current two-buffer scheme.

This memory estimate does not include velocity fields, PML memory arrays, receiver output, velocity model, or MPI/runtime overhead.
It only answers whether the additional pressure array itself is plausible.
